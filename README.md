# Refokus Clone | Work Section

A high-fidelity frontend clone of the Refokus agency website, built during my second semester. This project focuses on replicating high-end aesthetic layouts and smooth motion design.

## 🛠 Tech Stack
* **Framework:** React + Vite
* **Animations:** Framer Motion (for smooth transitions and scroll effects)
* **Styling:** CSS / Tailwind (whichever you used)

## 🚀 Key Features
* Replicated the "Work" section with precision.
* Integrated complex animations using Framer Motion variants.
* Responsive UI designed to look great on modern displays, like the HP OmniBook X.

## 📦 How to Run Locally
1. Clone this repo: `git clone https://github.com/AnshulOO2/Refokus-Clone.git`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
